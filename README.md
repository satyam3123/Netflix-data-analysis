# Netflix-data-analysis
This project performs exploratory data analysis (EDA) and data-driven insights generation on a Netflix movies and TV shows dataset containing 9,800+ titles. The goal is to understand content trends, genre distribution, popularity patterns, voting behavior, and release-year insights using Python-based data analytics and visualization.

The analysis covers data cleaning, feature engineering, categorical transformation, and visualization to answer key business-style questions about Netflix’s content strategy.
